parse_flagsFlags to set or unset options Flags available are

  • R_PARSE_HEADER_JWK
  • R_PARSE_HEADER_JKU
  • R_PARSE_HEADER_X5C
  • R_PARSE_HEADER_X5U

  • R_PARSE_UNSIGNED
  • R_PARSE_ALL

  • R_FLAG_FOLLOW_REDIRECT: follow redirections if necessary
  • R_FLAG_IGNORE_REMOTE: do not download remote key, but the function may return an error
